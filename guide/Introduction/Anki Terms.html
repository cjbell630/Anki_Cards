Definitions

Here are the meanings of some words I’m gonna use a lot here that will be helpful to know:

Deck

The main hub of Anki is made up of a collection of decks. Decks contain cards and can be studied.

Example

A deck containing cards about Japanese vocabulary.

Note

A note is a set of data about, for example, a vocabulary word. The data from notes are used to create cards. Each note can have multiple cards, but each card only pulls data from one note.

Example

A note about the word「古い」, containing fields “English”, “Kana”, and “Kanji”

Field

A field is a slot for one type of information on a note.

Example

On a note about「古い」, the field “English”, which contains the English meaning of the word

Card

A card is effectively just a real-life flashcard - one side has a question, and one side has the answer. 

Example

A card created from the note about the word「古い」which asks for the meaning
