Step 2: Creating your first note type

  • Step 2: Creating your first note type
    • Creating note types
Example


Once you have a deck, you need to create a note type. This is just a set of fields that you’ll use whenever you add a new word. The most basic setup would be to have a field for the word, and another for it’s translation. But that would defeat the whole point of using Anki!

For my main note type, I have lots of fields. This is just so it works with every vocab word I have, including verbs, nouns, etc. I don’t actually put text in all of these fields for every word, just the ones relevant to it.

I recommend, at the very least, having the following:

  • English: The English meaning
  • Kana: The word in kana (hiragana or katakana)
  • Kanji: The word with kanji (if it has any, otherwise I leave this empty)
  • Notes: Any extra information you want to add

To give an example of how you might use these four fields:

Field Example Value
English old
Kana ふるい
Kanji 古い
Notes Only used for things, not people

Creating note types

Now that you have an idea of what information you might want to include in your notes, here’s how you create a note type, starting from the home screen:

  1. Create a new empty note type:
    1. Click Add at the top of the screen
    2. Click the big button on the top towards the left next to Type (if you just downloaded the program, it should say “Basic” on the button)
    3. In the small window that popped up called Choose Note Type, click the Manage button on the bottom
    4. In the new small window called Note Types, click Add
    5. Yet another small window will pop up. Make sure Add: Basic is selected, then click OK
    6. Give it a name, maybe something like “Japanese Vocab”, and click OK
    7. Now, click Close in the Note Types window
    8. In the Choose Note Type window, make sure the new note type you just created is highlighted, then click Choose
    9. You should now be back to the Add screen, and there should be two text fields that say “Front” and “Back” above them.
  2. Add some fields:
    1. Click the Fields button towards the top left
    2. Small window, Fields for [your note type name].
    3. For the first two fields you want to add, highlight Front or Back, and click Rename on the right.
    4. After that, click Add on the right for any other fields you want to add. It might pop up with a warning about having to do a full upload. If it does, just hit Yes
    5. Once you’ve added all your fields, click Save at the bottom
IMPORTANT! English should be your first field. (click for more details)

I highly recommend making your first field the English one. This is because you cannot have multiple notes with the same value in the first field.

If you were to make, for example, “Kana” the first field, you would start having issues when adding any homophones, which Japanese has many of. With the English field, however, no two words should have the same value.

This is because, aside from the fact that no two words should have the EXACT same meaning (there will always be slight nuances), if they were the exact same you wouldn’t be able to distinguish between them when studying, so there’d be no point anyway.

That’s it! You’ve now created a note type, and can start creating card types.
