Step 1: Creating your first deck

  • Step 1: Creating your first deck
    • Creating a deck:
    • Creating subdecks
Example


The first thing you’ll want to do is create a deck. The simplest thing to do would be to make a single deck for all of your Japanese vocabulary and call it “Japanese”. But you can get much more organized if you like. For example, I have my main deck broken up into subdecks based on grammar type. There are also some sub-subdecks, such as “animals” in “nouns”. This is all personal preference, though.

Creating a deck:

  1. Click the button on the bottom of the home screen that says “Create Deck”
  2. Give it a name and hit “OK”

Creating subdecks

To create subdecks, you can either:

  • Type the name of another deck, followed by ::, followed by the name of the subdeck.
    • For example, to create the “adjectives” subdeck, I would create a new deck called master::adjectives.
  • Or, you can drag an existing deck below another deck, and let go.
